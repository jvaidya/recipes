Documentation of the home network at Rosalia.

Recipes

Jiten and Alicia's tasty recipes. You can edit and save these recipes.

HTTP daemon setup on 10.1.10.10 (aaibabaroompi)

I have installed tornado and mako. The http daemon script is at /storage/website/application.py. This is started on reboot from /storage/.config/startup.sh. No logs are currently written.

FTP daemon setup on 10.1.10.10 (aaibabaroompi)

I have installed pythonftpdlib. The ftp daemon script is at /storage/bin/openelecftpd. This is started on reboot from /storage/.config/startup.sh. No logs are currently written.
Username: guest, Password: guest, Directory: /media/mmcblk0p3-mmc-SU08G_0x1cb3022d/Scans which is exported using Samba.

Scan to FTP setup on 10.1.10.9 (BROTHER-MFC)

Using the web interface at http://10.1.10.9 I have set up the Scan To FTP function to use the FTP server on 10.1.10.10 above. You can scan to the profile 'aaibabaroompi' and the pdfs will be visible in the Scans folder via Samba.
